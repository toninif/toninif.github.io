Al compartir código:

  • se genera mayor transparencia

  • se puede comentar lo que se hizo y obtener feedback valioso

  • potencia el aprendizaje y la reproducibilidad de lo que hicimos

library(tidyverse) # En esta linea se carga el paquete que nos permite manejar datos
library(palmerpenguins) # En esta linea se carga el paquete con los datos

nuestrosDatos <- penguins %>% # En esta linea se cargan datos
  filter(species == "Adelie") # En esta linea se filtran datos
